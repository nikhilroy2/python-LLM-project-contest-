# Contributing to Judgmental Prediction Bot

This bot is designed to improve upon the [manifoldbot](https://github.com/microprediction/manifoldbot) package. Contributions that enhance both this bot and the upstream manifoldbot package are especially welcome.

## Areas for Contribution

### 1. Manifoldbot Improvements

Key improvements that could be contributed back to manifoldbot:

- **Enhanced Market Filtering**: The `MarketAnalyzer` class provides sophisticated market filtering and evaluation that could be useful in manifoldbot
- **Risk Management**: The `RiskManager` class offers portfolio-level risk controls that could benefit other bots
- **Performance Tracking**: The `PerformanceTracker` provides comprehensive analytics that could be a useful addition
- **LLM Integration**: The `LLMAnalyzer` demonstrates how to integrate LLMs for market analysis

### 2. Strategy Improvements

- New trading strategies
- Strategy optimization using backtesting
- Machine learning models for prediction
- Market sentiment analysis

### 3. Code Quality

- Better error handling
- More comprehensive tests
- Documentation improvements
- Type hints and type checking

## Pull Request Process

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Add tests if applicable
5. Update documentation
6. Submit a pull request

## Code Style

- Follow PEP 8
- Use type hints
- Write docstrings for all public functions and classes
- Keep functions focused and small
- Add comments for complex logic

## Testing

Before submitting a PR, ensure:
- Code runs without errors
- All existing tests pass
- New functionality is tested (if applicable)

## Questions?

Open an issue for questions or suggestions!
